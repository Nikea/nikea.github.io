UseCase HXN_DEV: Tomography with self-absorption correction

HXN

From Gabe Iltis's notes 6-25-14

For this task, we expect to do tomography based on different data, such as absorption, phase and fluorescence.

• An automatic alignment method needs to be developed for both vertical and horizontal directions.

• Self-absorption effect needs to be emphasized, and correction options need to be available.

• Current algorithms such as FBP, ART need to be considered.

This tool is being developed by Li Li and is 30% complete as of 6-25-14
