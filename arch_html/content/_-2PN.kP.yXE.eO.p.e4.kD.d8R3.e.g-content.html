UseCase HXN_DEV: XRF Microscopy

HXN

From Gabe's notes: 6-25-14

This analysis tool mainly performs the following tasks:

  1. Sum spectrum peak fitting: sum individual spectrum from each pixel and do fitting to determine which elements exist in the sample.

  2. Single pixel peak fitting to quantify the element in each 2D pixel

  3. Visualization:

    1. Overall spectrum with fitting

    2. Spectrum from individual pixel

    3. Spectrum over spatial ROI

    4. ROI element maps

    5. ROI fitted element maps

    6. Error maps, i.e. chi^2

This analysis tool is being developed by Li Li and is 90% complete, as of 6-25-14
